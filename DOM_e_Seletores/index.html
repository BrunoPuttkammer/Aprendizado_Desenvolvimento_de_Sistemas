O DOM (Document Object Model) é uma representação estruturada de um documento HTML ou XML em forma de uma árvorehierárquica. Ele permite que os navegadores e outras tecnologias interajam com os elementos da página de formaprogramática. Cada elemento da página é representado como um "nó" no DOM, com relacionamentos pai-filho que refletem a estrutura do documento.

Essa representação do documento permite que os desenvolvedores acessem, modifiquem e manipulem dinamicamente os elementos e conteúdos da página usando linguagens de programação, como JavaScript.
