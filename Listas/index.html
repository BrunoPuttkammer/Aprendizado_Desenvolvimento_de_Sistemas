Listas Ordenadas, Desordenadas e Descritivas

Listas Ordenadas:

Para criarmos uma lista ordenada, precisamos utilizar a tag <ol> ("ol" significa Ordered List),
e para cada item da lista utilizamos a tag <li>(List Item). Confira:

  1. Café
  2. Expresso
  3. Milkshake

Além disso, podemos personalizar as listas ordenadas utilizando as propriedades: "1"(padrão) "a", "A", "i" ou "I", Confira:

  1. Café
  2. Expresso
  3. Milkshake
  1. Café
  2. Expresso
  3. Milkshake

Listas Desordenadas:

Para criarmos uma lista desordenada, precisamos utilizar a tag <ul> ("ul" significa Unordered List),
e para cada item da lista utilizamos novamente a tag <li>(List Item). Confira:

  • Café
  • Expresso
  • Milkshake

Além disso, podemos personalizar as listas desordenadas utilizando as propriedades: "disc"(padrão),"circle","square" e "none", Confira:

  • Café
  • Expresso
  • Milkshake
  • Café
  • Expresso
  • Milkshake

Listas Descritivas:

Para criarmos uma lista descritiva, precisamos utilizar a tag <dl> ("dl" significa Description List),
e para cada item da lista utilizamos a tag <dt>(Define Term) seguida por uma tag <dd>(Describe Term). Confira:

Café
- Bebida preta,quente e amarga
Expresso
- Bebida marrom,quente e adoçicada
Milkshake
- Bebida clara,gelada e doce
