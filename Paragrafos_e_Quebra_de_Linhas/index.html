Para utilizarmos o elemento parágrafo, basta digitarmos um texto dentro das tags <p> e </p>.

Se você quiser digitar um texto com quebra de linhas, podemos utilizar a tag <br>, que significa "Break", para separar seus parágrafos,observe:
texto 1
texto 2
texto 3

Se quisermos quebrar a linha,para digitar um texto abaixo, podemos utilizar a tag <hr>, que significa "Horizontal Row",esta é uma tag auto fechante, e serve para criar uma linha horizontal na tela, que separa um elemento do outro.
